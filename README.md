# 🚀 ChatGPT Clone (JavaScript)

A simple **ChatGPT Clone** built using **HTML, CSS, and JavaScript**, powered by the **OpenAI API**. For now, I have used the GROQ API as this provides free version to use.
This project replicates the core functionality of ChatGPT, allowing users to interact with an AI chatbot in real-time.
For searching from the web, I have used Tavily in this project.
---

## 📌 Features

- 💬 Interactive chat interface  
- ⚡ Real-time AI responses  
- 🧠 Powered by OpenAI API  
- 📱 Responsive design  
- 🎨 Clean and minimal UI  
- 🔄 Continuous conversation support  

---

## 🛠️ Tech Stack

- HTML5 – Structure  
- CSS3 – Styling  
- JavaScript (Vanilla JS) – Logic  
- OpenAI API – AI responses  
- Node JS and Express for API development

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

git clone https://github.com/HemantKhatri004/Chatgpt-clone-js.git
cd Chatgpt-clone-js

### 2️⃣ Add your GROQ API Key

Open your JavaScript file (e.g., `script.js`) and replace:

const API_KEY = "your_groq_api_key_here";

---

### 3️⃣ Run the project

Open `index.html` in your browser.
