A full-stack **ChatGPT Clone** built using **HTML, CSS, and JavaScript**, powered by an AI API and a **Node.js + Express backend**.

This project simulates real-time conversational AI with **chat history support**, showcasing full-stack development and Generative AI integration.

---

## ✨ Features

- 💬 Real-time AI chat interface  
- 🧠 Chat history (session-based using NodeCache)  
- ⚡ Fast and optimized API handling  
- 🔐 Secure backend (API key protected)  
- 📱 Responsive design  
- 🎨 Minimal and clean UI  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Caching:** NodeCache (in-memory)  
- **API:** OpenAI API

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository
git clone https://github.com/HemantKhatri004/Chatgpt-clone-js.git  
cd Chatgpt-clone-js

---

### 2️⃣ Install dependencies
npm install

---

### 3️⃣ Configure environment variables

Create a `.env` file:

GROQ_API_KEY=your_groq_api_key_here

---

## ▶️ Run Locally

### 4️⃣ Start backend server
node -env-file=.env server.js

---

### 5️⃣ Start frontend
- Open index.html  
OR  
- Use Live Server in VS Code  

---
